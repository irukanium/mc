座標のペースト機能について

Java版 F3 + C で座標と角度をコピー
統合版 Ctrl + Alt + C で座標をコピー

コピー後、各項目のペーストボタンを押すと自動入力されます。

測量の注意点

  • 最低100マス以上間隔をあけて測量してください。
  • 座標やYawを入力する際は、小数点まで正確に入力してください。
  • チャンク内座標の4, 4にエンド要塞の入口が生成されます。
  • XかZの絶対値が約400以下の場所は、要塞が生成される確率は低いです。
  • 黄色の円は誤差の範囲を示しますが、円の範囲内に必ず要塞が存在するとは限りません。
